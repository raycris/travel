# Travel World
Landing page about traveling around the word, you can see hot trending stays, FAQ and recommendations. This landing page also is responsive and have dark mode as well, take a look in the pictures below. 

## Overview
- Framework - React
- Lenguage - JavaScript
- Styling - TailwindCSS
- Linting - ESLint
- Formatting - Prettier
